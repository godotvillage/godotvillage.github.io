## 简介

Alpha 是一个专为 Godot 编辑器打造的 AI Agent，让游戏开发者在编辑器内直接享受 AI 的强大能力。无需切换窗口，无需复制粘贴，你的创意想法可以在这里直接转化为游戏代码。Alpha 基于 DeepSeek 这一快速、准确、高效且高性价比的 AI 模型，为你提供最智能的开发体验。

![](/assets/images/alpha/overview.png)

## 下载并使用

### 资产库下载
- 在Godot中选择资产库
- 然后再在输入框中输入`AlphaAgent`，等待搜索结果
- 搜索出结果后，点击插件，然后点击下载
- 下载完成后，点击安装
- 安装后，即可看到`资产“AlphaAgent”安装成功！`字样
![](/assets/images/alpha/download.png)
![](/assets/images/alpha/install.png)
### 手动下载
如果因网络等原因无法访问Godot资产库，也可以手动下载。
- 访问 [AlphaAgent仓库发布页面](https://github.com/925236118/AlphaAgent/releases)
- 选择最新版本下载zip压缩包
- 解压后将addons文件夹复制到项目目录
- 完成手动下载

### 启用
- 在项目管理中选择插件
- 启用AlphaAgent
- 然后即可在编辑器右侧看到插件
![](/assets/images/alpha/enable.png)

## 使用指南

### 🚀 快速开始

激活插件后，Alpha 就会出现在右侧侧边栏中，随时准备为你服务。就像一位贴心的开发伙伴，时刻待命。

### 💬 开始对话

想要与 AI 交流？只需在 Alpha 下方的输入框中输入你的问题，点击发送按钮，Alpha 就会立即响应。无论是代码问题、功能实现，还是创意想法，Alpha 都能为你提供专业的帮助。

![](/assets/images/alpha/chat.png)

### 🎯 模式选择

Alpha 为你准备了两种智能模式：**Ask** 与 **Agent**，默认开启 Agent 模式，让你体验最强大的功能。

- **Agent 模式**：Alpha 的"超级模式"，可以调用引擎内的各种工具，实时获取当前打开文件的状态、读取文件内容、修改代码等。就像一位能够直接操作编辑器的 AI 助手，让开发效率倍增。
- **Ask 模式**：纯对话模式，专注于问答交流，不调用任何工具。适合进行技术咨询、概念讨论或获取建议的场景。

两种模式都支持思考功能，你可以在设置中开启或关闭。

![](/assets/images/alpha/mode.png)

### 📎 文件引用

想让 Alpha 更准确地理解你的代码？只需将文件从编辑器中直接拖动到输入框内，Alpha 就能立即"看到"文件内容。

目前支持从以下位置拖入：
- 文件系统侧边栏
- 场景树
- 脚本编辑器侧边栏

支持拖入文件、文件夹或节点。发送后，Alpha 会优先读取这些文件的内容，为你提供更加精准的回答。

![](/assets/images/alpha/reference.png)

### 📚 历史记录

每一次与 Alpha 的对话都会被完整记录。点击右上角的历史记录按钮，即可回顾所有对话。

- **恢复对话**：点击恢复按钮，可以重新打开某次对话，继续之前的讨论。
- **删除记录**：点击删除按钮可以永久删除对话记录（此操作无法撤销，请谨慎操作）。

历史记录保存在项目文件内，如果你需要在不同设备间同步，只需上传 `addon/agent` 目录下的更新即可。

![](/assets/images/alpha/history.png)

### 🧠 记忆功能

Alpha 拥有强大的记忆能力，可以记住你的重要信息。

使用方法很简单：在输入框内键入 `/memory`，然后指定存储位置（`global` 全局记忆或 `project` 项目记忆），再输入要保存的内容，发送即可。

**示例**：输入 `/memory project 我是xxx`，保存成功后，当你问 Alpha"我是谁？"时，它就能准确回答。

> **注意**：如需删除记忆，请前往 `res://addons/agent/config.tres` 文件中手动删除。

### ⚙️ 设置

Alpha 提供了丰富的自定义设置选项，让你可以根据自己的需求调整 AI 的行为。

**进入设置**：点击 Alpha 面板右上角的设置按钮，即可打开设置界面。

**可配置选项**：

- **思考功能**：开启后，AI 在回答问题时会展示思考过程，让你了解 AI 是如何分析和推理的。
- **自动展开思考内容**：开启后，思考内容会自动展开显示；关闭则默认折叠，点击可展开查看。
- **自定义密钥**：支持配置你自己的 API 密钥，享受更灵活的使用体验。[购买DeepSeek平台Token教程](https://www.moshangzhu.com.cn/work/%E8%B4%AD%E4%B9%B0DeepSeek%E5%B9%B3%E5%8F%B0Token.html)

> **提示**：Agent 模式下调用工具时也支持思考模式，你可以查看工具的请求和返回内容，更好地理解 AI 的工作过程。
