# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2025-12-04

### ✨ New Features (新增功能)
- **Settings**:
  - 添加设置界面和全局Setting配置系统。
  - 添加是否使用思考功能的配置选项。
  - 添加是否要自动展开思考内容选项
  - 添加自定义密钥选项
- **AI Assistant**:
  - 添加用户输入框内回车发送消息功能。
  - Agent调用工具支持思考模式。
  - 可展示调用工具的请求和返回内容。
- **Tools**:
  - 优化查询文件目录信息，可查看部分目录，在大项目中减少性能消耗和卡死。
  - 添加获取图片信息工具。
  - 添加设置自动加载工具（支持设置或删除）。
  - 添加编辑器修改文件工具
  - 添加调用编辑器打开资源工具
  - 添加检查脚本语法错误工具

### 🐛 Bug Fixes (问题修复)
- 修复添加记忆后需要新开一个对话的问题。
- 修复恢复历史记录时工具调用结果未恢复的问题。
- 修复调用get_image_info阻塞问题。

### 🔧 Improvements (优化改进)
- 添加Agent面板最小宽度。
- 更新工具和错误处理机制。
- 更新提示词。
- 删除无关文件，优化项目结构。

## [0.1.0] - 2025-12-01

### ✨ New Features (新增功能)
- **AI Assistant**:
  - 添加对话功能以及Ask、Agent模式。
  - 添加了生成对话标题的功能。
  - 完成了历史记录功能 (History Container)，支持查看及管理过往对话。
  - 添加了记忆功能，提升上下文理解能力。
  - 引入了配置文件系统，用于管理插件设置。
- **Tools**:
  - 新增脚本工具，辅助开发流程。添加7种工具。
- **UI**:
  - 新增主面板UI以及界面。
  - 新增帮助页面。
  - 主面板添加了跳转至官方文档/页面的链接。
